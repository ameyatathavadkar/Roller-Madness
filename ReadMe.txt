The exe file can be found in the builds folder.You can either run it on a windows device or on your localhost server.
In order to run it on your device itself simply run the unity exe file in windows folder.
In order to run it on a local server run the index file from your sever.

Objectives-
To collect as many coins as you can

Movements-
Use WASD keys to navigate your ball.
You can either keep avoiding your enemies or kill them by making them fall off or by making them collide with each other.
